# ClinIQ
Multimodal RAG pipeline for clinical case retrieval in neuro-oncology. Combines radiomic feature extraction (PyRadiomics) with clinical note embeddings to enable hybrid vector search across brain tumor cases. Implements dual-vector retrieval using named vector collections in Qdrant, with Reciprocal Rank Fusion (RRF) for cross-modal re-ranking. Ingestion pipeline handles NIfTI medical imaging, automated binary mask preprocessing, and feature normalization before vector storage. Query pipeline accepts imaging features, natural language clinical descriptions, or both — returning the most similar historical cases ranked by combined radiomic and semantic similarity.
